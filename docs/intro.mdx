# Agent Hub

Agent Hub is an AI agent platform that lets you build, manage, and deploy AI agents. Each agent has its own identity, workspace, and tools - working like autonomous team members.

## Quick Start

Get up and running in minutes:

```bash
# 1. Create an agent via API
curl -X POST https://api.singularitybridge.net/api/assistants \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "my-agent",
    "llmProvider": "openai",
    "llmModel": "gpt-5.1",
    "systemPrompt": "You are a helpful assistant."
  }'

# 2. Execute a request
curl -X POST https://api.singularitybridge.net/api/assistant/{id}/execute \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"userInput": "Hello, what can you do?"}'
```

Or use the [MCP Server](/api/mcp-server) to manage agents from Claude, ChatGPT, or any MCP-compatible client.

## Key Features

- **Multi-LLM Support** - Use OpenAI, Anthropic Claude, Google Gemini, or any supported provider
- **MCP Protocol** - Native Model Context Protocol support for tool integration
- **25+ Integrations** - Connect to JIRA, GitHub, Perplexity, ElevenLabs, and more
- **Teams** - Organize agents into collaborative teams
- **Cost Tracking** - Monitor usage and costs across all agents
- **Prompt History** - Version control for agent prompts
